# Typescript SDK

> This SDK is fully compatible with Javascript as well.

## Installation

### npm

```bash
npm install @differentialhq/core
```

### yarn

```bash
yarn add @differentialhq/core
```

## Usage

```ts
import { Differential, PoolConfig } from "@differentialhq/core";

export const d = new Differential(
  "API_SECRET",
);
```
